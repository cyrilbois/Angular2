# Get started with Ionic Framework  

http://ionicframework.com/getting-started/  

```sh
$ npm install -g cordova ionic  
$ ionic start --v2 myApp blank
$ ionic start --v2 myApp tabs
$ ionic start --v2 myApp sidemenu
$ cd myApp
$ ionic serve
```
